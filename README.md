# 💳 Payment Gateway Integration

A full-stack **Payment Gateway Integration** project built using **React.js**, **Spring Boot**, and **Razorpay**. This project demonstrates how to integrate a secure online payment gateway by connecting a React frontend with a Spring Boot backend.

---

## 🚀 Features

* 💳 Razorpay Payment Gateway Integration
* ⚛️ React.js Frontend
* ☕ Spring Boot Backend
* 🔗 REST API Communication
* 📦 Order Creation using Razorpay API
* ✅ Payment Success & Failure Handling
* 🧪 Razorpay Test Mode Support
* 🔒 Clean Project Structure

---

## 🛠️ Tech Stack

### Frontend

* React.js
* Axios
* CSS

### Backend

* Spring Boot
* Spring Web
* Spring Data JPA
* MySQL
* Razorpay Java SDK

### Payment Gateway

* Razorpay Checkout
* Razorpay Orders API

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/YOUR_REPOSITORY.git
```

### Frontend

```bash
cd payment-frontend
npm install
npm run dev
```

### Backend

```bash
cd payment-backend
mvn spring-boot:run
```
